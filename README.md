# 🌱 OptiCrop – Smart Agricultural Production Optimization Engine

OptiCrop is an AI-powered crop recommendation system that predicts the most suitable crop based on soil nutrients and environmental conditions using Machine Learning. The application helps farmers make informed agricultural decisions by providing crop recommendations along with cultivation details and farming tips.

## 📌 Project Overview

OptiCrop is developed to recommend the most suitable crop using machine learning based on the following agricultural parameters:

- Nitrogen (N)
- Phosphorus (P)
- Potassium (K)
- Temperature
- Humidity
- Soil pH
- Rainfall

The application predicts one of **22 crop types** and displays crop-specific information including growing season, ideal temperature, water requirements, and cultivation tips.

---

## ✨ Features

- AI-based crop recommendation
- Predicts among 22 different crops
- Modern and responsive web interface
- Crop images and cultivation details
- Farming tips and environmental requirements
- Real-time prediction using a trained Random Forest model

## 🚀 Run the Project Locally

### Clone the Repository

```bash
git clone https://github.com/khanafiatasleem/OptiCrop-Smart-Agricultural-Production-Optimization-Engine.git
```

### Move to the Project Folder

```bash
cd OptiCrop-Smart-Agricultural-Production-Optimization-Engine
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Flask Application

```bash
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```
